# 官方文件 URL 結構

> 查閱文件時的快速參考。URL 可能隨版本更新變動，發現變更時請更新此文件。

## Angular 官方 (angular.dev)

| 類別 | URL Pattern | 說明 |
|------|-------------|------|
| 總覽 | `https://angular.dev/overview` | Angular 概觀 |
| API 參考 | `https://angular.dev/api` | 完整 API 文件 |
| 元件 | `https://angular.dev/guide/components` | Component 指南 |
| Signals | `https://angular.dev/guide/signals` | Signal 系統 |
| 路由 | `https://angular.dev/guide/routing` | Router 指南 |
| 表單 | `https://angular.dev/guide/forms` | Forms 指南 |
| HTTP | `https://angular.dev/guide/http` | HttpClient 指南 |
| DI | `https://angular.dev/guide/di` | 依賴注入 |
| 測試 | `https://angular.dev/guide/testing` | 測試指南 |

## Angular Material 官方 (material.angular.io)

| 類別 | URL Pattern | 說明 |
|------|-------------|------|
| 元件總覽 | `https://material.angular.io/components/categories` | 所有 Material 元件 |
| 特定元件 | `https://material.angular.io/components/{name}/overview` | 如 `button`、`dialog` |
| 元件 API | `https://material.angular.io/components/{name}/api` | 元件 API 文件 |
| CDK | `https://material.angular.io/cdk/categories` | CDK 功能列表 |
| 主題指南 | `https://material.angular.io/guide/theming` | Material 3 主題設定 |

## GitHub Source

| 類別 | URL |
|------|-----|
| Angular Core | `https://github.com/angular/angular` |
| Angular Material | `https://github.com/angular/components` |

## WebSearch 建議關鍵字模板

```
# Angular 核心功能
"Angular 21 {feature} site:angular.dev"

# Material 元件
"Angular Material 21 {component} site:material.angular.io"

# 遇到問題時
"Angular 21 {error-message-or-topic} site:angular.dev OR site:github.com/angular"
```
